This repository contains my geospatial data science and GeoAI projects, covering satellite image classification, environmental monitoring, and spatial analysis. All projects are implemented in **Python** using **PyTorch**, **Google Earth Engine**, **GeoPandas**, and related geospatial libraries.

---

## 📂 Projects

### 🔬 [`Deep_Learning/`](./Deep_Learning)
**Satellite Image Classification with CNNs**
- Built and trained convolutional neural networks (CNNs) using **PyTorch** for land cover classification from **Sentinel-2** multispectral imagery.
- Developed a custom **1D-CNN model** with block cross-validation for vegetation health mapping.
- **Tools:** PyTorch, NumPy, Scikit-Learn, Matplotlib

### 🌊 [`GRADCAM/Flood Mapping/`](./GRADCAM/Flood%20Mapping)
**Explainable AI for Flood Mapping**
- Applied deep learning models for flood-prone area detection from satellite imagery.
- Utilized **Grad-CAM** (Gradient-weighted Class Activation Mapping) visualizations to provide interpretability for model predictions, highlighting which spatial features drive classification decisions.
- **Tools:** PyTorch, Grad-CAM, Matplotlib

### 🛰️ [`GeoAI/`](./GeoAI)
**GeoAI for Object Detection and Feature Extraction**
- Implemented geospatial AI workflows for object detection from satellite and aerial imagery.
- Applied **SAMGeo** (Segment Anything Model for Geospatial Data) for automated building footprint extraction from high-resolution imagery.
- **Tools:** SAMGeo, TorchGeo, Google Earth Engine

### 🌿 [`REMOTE SENSING/`](./REMOTE%20SENSING)
**Remote Sensing for Environmental Monitoring**
- Processed and analyzed remote sensing data for biomass estimation and environmental monitoring.
- Worked with multi-temporal satellite imagery for landscape change detection.
- **Tools:** Google Earth Engine, GeoPandas, Geemap

### 📘 [`Basics/`](./Basics)
**Geospatial Fundamentals**
- Foundational GIS and remote sensing exercises covering spatial analysis, coordinate systems, and map projections.
- **Tools:** ArcGIS Pro, QGIS, Python
